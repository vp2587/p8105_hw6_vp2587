**The bootstrap is helpful when you‚Äôd like to perform inference for a
parameter / value / summary that doesn‚Äôt have an easy-to-write-down
distribution in the usual repeated sampling framework. We‚Äôll focus on a
simple linear regression with tmax as the response with tmin and prcp as

estimated from these data:**

- ùëüÃÇ 2

- ùõΩÃÇ 1ùõΩÃÇ 2

**Use 5000 bootstrap samples and, for each bootstrap sample, produce
estimates of these two quantities. Plot the distribution of your
estimates, and describe these in words. Using the 5000 bootstrap
estimates, identify the 2.5% and 97.5% quantiles to provide a 95%
confidence interval for ùëüÃÇ 2 and ùõΩÃÇ 1ùõΩÃÇ 2.**

**Note: broom::glance() is helpful for extracting ùëüÃÇ 2 from a fitted
regression, and broom::tidy() (with some additional wrangling) should
help in computing ùõΩÃÇ 1ùõΩÃÇ 2.**

``` r
set.seed(123)
weather_boot = 
  weather_df %>% 
  drop_na(tmax, tmin, prcp) %>%
  bootstrap(n = 5000, id = "strap_id") %>% 
  mutate(
    fit = map(strap, ~lm(tmax ~ tmin + prcp, data = .x)),
    glance = map(fit, broom::glance),
    tidy = map(fit, broom::tidy)
  ) %>% 
  mutate(
    r2_weather = map_dbl(glance, "r.squared"),
    beta_ratio = map_dbl(tidy, ~ {
      b1 = .x %>% 
        filter(term == "tmin") %>% pull(estimate)
      b2 = .x %>% 
        filter(term == "prcp") %>% pull(estimate)
      b1 / b2
    })
  ) %>% 
  select(strap_id, r2_weather, beta_ratio)

weather_long =
  weather_boot %>% 
  pivot_longer(
    cols = c("r2_weather", "beta_ratio"),
    names_to = "parameter",
    values_to = "estimate"
  )

weather_long %>% 
  ggplot(aes(x = estimate)) + 
  geom_histogram(bins = 50) +
  facet_wrap(. ~ parameter, scales = "free") +
  labs(
    title = "Bootstrap distributions for r^2 and Œ≤1/Œ≤2",
    x = "Bootstrap estimate",
    y = "Count"
  ) +
  theme_minimal()
```
